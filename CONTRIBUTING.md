# Contributing

## Branching
- main: stabilno
- feat/*: nove funkcionalnosti
- fix/*: ispravke

## Commits
- Koristi konvencije tipa: feat:, fix:, docs:, chore:
- Primer: `feat(plugin): dodaj osnovni skeleton za search plugin`

## PR pravila
- Mali, fokusirani PR-ovi
- Opis: šta, zašto, kako testirati
- Prolaz CI (lint + phpstan)
